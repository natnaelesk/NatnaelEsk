### Now you owe me a star on one of my favorite projects and open sources

- **[AutoJobs](https://github.com/natnaelesk/job-auto-applier)** — by far the project I'm most proud of. It helped me land my 2nd job very easily. It's an automation tool built on multiple agents. The first 4 agents collect job posts from different Telegram channels, filter the ones that suit me best, add them to Notion, and polish my CVs. So at some point in the day I just sit down, start the 5th agent, it takes me to the link, I fill the form and upload my CV — done. The 6th agent scans my Gmail for potential replies and tells me. So cool. I built it with the Cursor SDK and Python. Absolute banger.

- **[DeskExercise](https://github.com/natnaelesk/deskExercise)** — this is my favorite project. I made it because I sit for very long hours. It's a desktop app that reminds you to get up and exercise, but deeper: it can block usage, customize the timing, all of that. Nothing crazy — just a 2-hour vibe-coded project. I love it because of how useful it is for me.

- **[Hiriya Chatbot](https://github.com/natnaelesk/hiriya_ChatBot)** — this is the project I'm still mad at for not being in production. It's a smart chatbot for Ambo University with a vector database and RAG system to pull knowledge. Me and my team put a lot of work into that knowledge base — basically everything about the university: offices, workflows, city navigation, campus navigation, recommendations. Just a cool project.

- **[YouTube Automation](https://github.com/natnaelesk/youtube-automation)** — love this one because it was the first project I got scammed by a client on (stupid me). I built it in a public repo and the client ditched me. Never mind — still love it. Cool project. It uses an LLM (Grok) to analyze YouTube videos and tell if there's a sponsor in the video or on the channel.
